Calculus 1 — Complete Cheat Sheet

Quick reference: limits, derivatives, integrals, differentiation rules, applications, and optimization.

Limits — Definition & Notation

Definition: limx→a f(x) = L means f(x) gets arbitrarily close to L as x approaches a.

Concept Definition / Example
Left Limit limx→a− f(x)
Right Limit limx→a+ f(x)
Limit Exists Left and right limits exist and are equal
Limit at Infinity limx→±∞ f(x) = behavior as x gets very large
Infinite Limit limx→a f(x) = ±∞ (vertical asymptote)

Key: A limit at a point doesn't require the function to be defined there.

Limit Laws

If limx→a f(x) = L and limx→a g(x) = M, then:

Law Formula
Sum lim [f(x) + g(x)] = L + M
Difference lim [f(x) − g(x)] = L − M
Product lim [f(x) · g(x)] = L · M
Quotient lim [f(x)/g(x)] = L/M (if M ≠ 0)
Power lim [f(x)]n = Ln
Root lim [f(x)]1/n = L1/n (if L > 0, n even)
Constant Multiple lim k·f(x) = k·L

Continuity

Definition: f is continuous at x = a if:

  • f(a) is defined
  • limx→a f(x) exists
  • limx→a f(x) = f(a)
Type Description
Removable Discontinuity Limit exists but f(a) ≠ limit, or f undefined (hole in graph)
Jump Discontinuity Left and right limits exist but are not equal
Infinite Discontinuity Vertical asymptote; limit is ±∞
Oscillating Discontinuity Function oscillates without approaching a value (e.g., sin(1/x) at x=0)

Intermediate Value Theorem (IVT): If f is continuous on [a,b] and N is between f(a) and f(b), then ∃ c ∈ (a,b) where f(c) = N.

Derivatives — Definition & Notation

Definition: The derivative of f at x is the slope of the tangent line:

f'(x) = limh→0 [f(x+h) − f(x)] / h
Notation Meaning
f'(x) Lagrange notation (prime)
df/dx Leibniz notation
D_x f Operator notation
f''(x) Second derivative (rate of change of f')
f^(n)(x) nth derivative

Geometric Interpretation: f'(a) is the slope of the tangent line to f at x = a.

Rule Formula Example

Critical Points & First Derivative Test

Critical Point: Where f'(x) = 0 or f'(x) undefined (potential max/min).

Test / Concept Description
First Derivative Test Check sign of f' before and after critical point:
+ to − → local max
− to + → local min
no sign change → inflection point
Increasing/Decreasing f'(x) > 0 on interval → f increasing
f'(x) < 0 on interval → f decreasing
Second Derivative Test At critical point c: f''(c) > 0 → local min; f''(c) < 0 → local max; f''(c) = 0 → test inconclusive
Concavity f''(x) > 0 → concave up ⌢
f''(x) < 0 → concave down ⌣
Inflection Point Where concavity changes; f''(x) = 0 (and sign changes)

Mean Value Theorem & Rolle's Theorem

Rolle's Theorem: If f is continuous on [a,b], differentiable on (a,b), and f(a) = f(b), then ∃ c ∈ (a,b) where f'(c) = 0.

Mean Value Theorem (MVT): If f is continuous on [a,b] and differentiable on (a,b), then ∃ c ∈ (a,b) where:

f'(c) = [f(b) − f(a)] / (b − a)

Interpretation: There exists a point where the instantaneous rate of change equals the average rate of change.

L'Hôpital's Rule

If limx→a f(x)/g(x) gives 0/0 or ±∞/±∞, then:

limx→a f(x)/g(x) = limx→a f'(x)/g'(x)

Process:

  • Verify indeterminate form (0/0 or ∞/∞)
  • Differentiate numerator and denominator separately
  • Evaluate the new limit
  • Repeat if necessary

Note: This is NOT the quotient rule! Only applies to indeterminate forms.

Integration — Definition & Notation

Definition: The antiderivative F of f satisfies F'(x) = f(x).

∫ f(x) dx = F(x) + C
Type Notation & Meaning
Indefinite Integral ∫ f(x) dx
Definite Integral ∫ab f(x) dx
Fundamental Theorem (Part 1) If F'(x) = f(x), then ∫ab f(x) dx = F(b) − F(a)
Fundamental Theorem (Part 2) d/dx [∫ax f(t) dt] = f(x)

Key: C (constant of integration) only appears in indefinite integrals.

Integration Rules

Rule Formula
Power ∫ xn dx = xn+1/(n+1) + C (n ≠ −1)
Sum ∫ [f+g] dx = ∫ f dx + ∫ g dx
Constant Multiple ∫ k·f dx = k ∫ f dx
1/x ∫ 1/x dx = ln|x| + C
ex ∫ ex dx = ex + C
ax ∫ ax dx = ax/ln(a) + C
sin(x) ∫ sin(x) dx = −cos(x) + C
cos(x) ∫ cos(x) dx = sin(x) + C

Integration Techniques

Technique When to Use Formula / Process
U-Substitution Composite functions (chain rule backward) Let u = inner function, du = u' dx, then substitute
Integration by Parts Product of functions (polynomial × trig, ln × x, etc.) ∫ u dv = uv − ∫ v du
(choose u using LIATE)
Partial Fractions Rational functions (polynomial ratios) Decompose into simpler fractions, then integrate each
Trig Substitution Expressions with √(a²−x²), √(a²+x²), √(x²−a²) √(a²−x²) → x = a·sin(θ)
√(a²+x²) → x = a·tan(θ)
√(x²−a²) → x = a·sec(θ)

Area Under Curve & Applications

Concept Formula / Description
Area Under f(x) A = ∫ab f(x) dx (f(x) ≥ 0)
Area Between Curves A = ∫ab [f(x) − g(x)] dx (f ≥ g)
Volume (Disk Method) V = π ∫ab [f(x)]² dx
Volume (Washer Method) V = π ∫ab {[f(x)]² − [g(x)]²} dx
Arc Length L = ∫ab √(1 + [f'(x)]²) dx
Surface Area S = 2π ∫ab f(x)√(1 + [f'(x)]²) dx

Optimization — Solving Max/Min Problems

Process:

  1. Set up the function to optimize: f(x) = what you want to maximize/minimize
  2. Identify constraints and express all variables in terms of one variable
  3. Find critical points: solve f'(x) = 0
  4. Use second derivative test or first derivative test to classify
  5. Check endpoints and compare all critical/endpoint values
  6. State the answer with units

Common Scenarios: Fence/enclosure problems, container optimization, cost minimization, revenue/profit maximization, projectile motion.

Absolute vs Local Extrema

Type Definition Where to Check
Local Maximum f(c) ≥ f(x) for all x near c Critical points where f' changes + to −
Local Minimum f(c) ≤ f(x) for all x near c Critical points where f' changes − to +
Absolute Maximum Largest value on entire domain Compare all critical points AND endpoints
Absolute Minimum Smallest value on entire domain Compare all critical points AND endpoints

Extreme Value Theorem (EVT): Continuous function on closed interval [a,b] attains absolute max/min.

Common Mistakes & Quick Tips

Mistake Correct Approach
Forgetting chain rule Always check for composition: d/dx[f(g(x))] = f'(g(x))·g'(x)
Quotient rule misapplication Remember: (u/v)' = (u'v − uv')/v², not u'/v'
Forgetting "+C" in indefinite integrals Always add constant of integration for antiderivatives
Confusing f(x) and f'(x) f'(x) = 0 → critical points; f'(x) > 0 → increasing; f'(x) < 0 → decreasing
Wrong limit at infinity reasoning Highest degree terms dominate; compare leading coefficients for rational limits
Missing negative in derivative formulas d/dx[cos(x)] = −sin(x); ∫ sin(x) dx = −cos(x) + C

Asymptotes

Type Definition & How to Find Example
Vertical x = a where lim f(x) = ±∞. Find where denominator = 0 (and numerator ≠ 0) f(x) = 1/(x−2) → x = 2
Horizontal y = L where limx→∞ f(x) = L. Compare degree of numerator and denominator for rational functions f(x) = (2x+1)/(3x−5) → y = 2/3
Oblique (Slant) y = mx + b when degree(numerator) = degree(denominator) + 1. Use polynomial long division f(x) = (x²+1)/(x−1) → y = x+1

Related Rates Problems

Process:

  1. Draw a diagram and label variables/constants
  2. Identify what's given (rate) and what to find (another rate)
  3. Write an equation relating the variables using geometry/physics
  4. Differentiate both sides with respect to time: d/dt
  5. Substitute known values and solve for unknown rate

Key Insight: All rates are with respect to time. Use implicit differentiation!
